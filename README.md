# 🏥 Healthcare Data Analytics Dashboard (Power BI)

## 📊 Overview

This project presents an end-to-end healthcare data analytics solution developed in Power BI, focused on analysing smoking-related health risks and patient demographics.

The objective was to transform raw healthcare data into structured insights that support data-driven decision-making, risk identification, and improved understanding of patient behaviour and outcomes.

---

## 🎯 Objectives

* Analyse the relationship between smoking habits and health risks
* Identify high-risk patient groups using demographic data
* Build an interactive dashboard for healthcare insights
* Enable clear visual storytelling for non-technical stakeholders

---

## 🛠️ Tools & Technologies

* **Power BI** (Dashboard Development & Visualisation)
* **Power Query** (Data Cleaning & Transformation)
* **DAX** (Calculated Measures & KPIs)
* **Data Modelling** (Relationships & Structured Datasets)

---

## 📈 Key Features

* Interactive dashboard with dynamic filtering and drill-down capabilities
* Clean and structured data model built from raw healthcare datasets
* Custom DAX measures for health risk indicators and trend analysis
* Visualisations combining demographic and behavioural data
* User-friendly design tailored for decision-making and insights

---

## 🔍 Key Insights

* Clear correlation identified between smoking frequency and increased health risk levels
* Certain demographic groups (age/gender) show higher exposure to health risks
* Trends highlight patterns in patient behaviour that may inform preventive strategies
* Data visualisations simplify complex healthcare information into actionable insights
